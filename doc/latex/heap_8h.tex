\subsection{Detailed Description}
Simple definition of a binary heap containing {\ttfamily floats} values. \begin{DoxyAuthor}{Author}
Gornet MÃ©lanie
\end{DoxyAuthor}
This is a simple definition of a binary heap containing {\ttfamily floats} values implemented using a binary tree. Warning\-: the cells pointers should not be shared as deallocation of a B\-S\-T deallocates the memory space reserved for the cells!

Inserting a value and extracting the minimum value are guaranteed to have a worst-\/case complexity in \$\-O( n)\$ where \$n\$ is the size of the heap.

The A\-P\-I of the binary heap is defined as follows\-:


\begin{DoxyItemize}
\item a function to create an empty heap
\item a function to check if the heap is empty
\item a function to insert a new value in the heap
\item a function to get the minimum value in the heap
\item a function to print the binary heap
\item a function to deallocate the binary heap 
\end{DoxyItemize}

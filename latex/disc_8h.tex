\subsection{Detailed Description}
Draw D\-I\-S\-Cs with S\-D\-L. This module exposes a few functions to ease the use of the S\-D\-L library, and a D\-I\-S\-C drawing routine.

Use -\/lm -\/l\-S\-D\-L when compiling in order to link with math library and S\-D\-L library.
